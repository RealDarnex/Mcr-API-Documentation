About

Mcr API is an API designed for Minecraft development. The project is developed by Mcr Studio.

It was created on September 13, 2025, in Germany by Darnex.

How to Use This Documentation

-------- Sidebar Navigation --------

The sidebar is located on the left side of the page and contains four navigation buttons:

Blue Face Icon: Redirects you to our Discord community.

Red Box with Play Button: Takes you to our YouTube channel.

Black Circle with White Cat: Links to our source code and examples on GitHub.

Black Box with Music Note Logo: Redirects you to our TikTok profile.

-------- Search Bar --------

The search bar is located at the top of the page, just below the header. You can use it to quickly find methods by typing their names. The search will suggest matching methods along with their type (e.g., Entity, World, etc.). Clicking on a suggestion will take you directly to the corresponding documentation page, where you can find detailed information about usage, functionality, and other important details. You can also search for specific error codes using the format "Error: %number%".
